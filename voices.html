Voice Gallery

Read‑Aloud lists the text-to-speech voices already installed on your device. Here is why lists differ by browser and how to add more voices on every major platform.

Why voice lists look different

Read‑Aloud uses the Speech Synthesis API built into your browser. That API surfaces the voices shipped with your operating system plus any optional packs you have installed. macOS and iOS often ship with several natural voices, while some Windows builds include only a handful until you add language packs. ChromeOS and many Android devices rely on Google’s system TTS engine, which exposes different options depending on region and device manufacturer.

The browser also matters. Chrome and Edge often expose the newest voices first, while Firefox may list fewer voices on some platforms. If you switch browsers and see additional options, that is expected: each browser filters the system catalog differently. Regardless of the list, all playback happens locally—the tool does not stream your text to a cloud engine.

How to install or enable voices

iOS and iPadOS

  • Open Settings → Accessibility → Spoken Content and tap Voices.
  • Select a language, then download an enhanced voice such as “Siri Voice 4.” Enhanced voices sound more natural but may take longer to download.
  • Return to Read‑Aloud and reload the page. The new voice should appear in the list after the download finishes.

macOS

  • Open System Settings → Accessibility → Spoken Content and choose System Voice → Manage Voices.
  • Add the languages and specific voices you want, then allow downloads to complete.
  • Restart the browser to refresh the Speech Synthesis catalog.

Windows 10/11

  • Open Settings → Time & Language → Speech and review the Manage voices section.
  • Use Add voices to install language packs that include TTS voices. Choose voices that match the language you paste most often.
  • Restart the browser; Edge and Chrome tend to expose new voices immediately after a reboot.

Android

  • Open Settings → System → Languages & input → Text-to-speech output.
  • Select Preferred engine (often Google’s engine) and tap the gear icon to manage installed voices.
  • Download offline voice data for the languages you use. Some manufacturers hide voices under “Offline speech recognition.”
  • Reopen Read‑Aloud in Chrome. If the voice list is still short, try enabling “Use network voices” in the same settings panel.

What to do if voices are missing

If the list still looks sparse after installing voices, try these quick fixes:

  • Close and reopen the browser. Some platforms only refresh the list on launch.
  • Disable battery saver or low-power modes that might pause voice downloads mid-way.
  • On desktop, open a private window to rule out extensions blocking the Speech Synthesis API.
  • Ensure your system language settings match the voice language you want; mismatched locales sometimes hide voices.

Still stuck? Send your device model, OS version, browser version, and a screenshot of the empty dropdown through the Contact page. We maintain a running list of browser quirks and can suggest a specific pairing that works for your setup.
