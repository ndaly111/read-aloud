Support

Start with the Help Center for step-by-step fixes. If you still need assistance, send a detailed report.

Self-serve resources

  • Help Center — platform-specific checklists for no sound, missing voices, and offline use.
  • Voice Gallery — instructions for adding system voices on iOS, macOS, Windows, and Android.
  • Guides — full workflows for students, writers, and accessibility needs.

Tip: keep the iPhone/iPad checklist bookmarked if you switch devices often.

Contact us

Use the Contact page and include browser, device, what happened, and a short snippet of the text length you tried. Most replies arrive within two business days.

If you are reporting a classroom or accessibility issue, mention deadlines or accommodation details so we can prioritize correctly.

For billing questions about premium MP3s, include your transaction ID and the voice you selected. Technical questions about the free tool are handled first to keep the site reliable for everyone.

Service status

Read‑Aloud is a static site; outages are rare. If the page fails to load, check your network or try a different browser profile. Known issues, if any, will be posted on the Product Updates page.

If audio works elsewhere but not here, see the platform checklists before writing in.

For campus networks that block audio streams, switch to mobile data for the first playback to grant audio permission.
